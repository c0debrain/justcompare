Hi! I am Chaitanya. JustCompare is a price comparison engine that I created as a project. Its primary purpose as the name suggests is to provide a web interface which allows users to compare prices of books.

It gathers information using web scrapers which were programmed specifically to gather book prices and other details from leading Indian retailers. Once the data is gathered it is stored in a postgresql database from where it is fetched for display on the website.

The scrapers were written in python. Web.py was used as the web framework for the app and it is currently hosted using the same with apache and mod_wsgi. The code and database dump is available on Github.

Because its been a while since the scrapers were run (four months to be precise), the prices of books on the seller sites have changed. However I am planning on running the scrapers again around the 20th of December (once my exams are over) so be sure to check again!
